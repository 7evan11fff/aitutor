üîç Step-by-Step Troubleshooting:

Step 1: Try Aggressive Mode

If normal clicking doesn't work, try aggressive mode:

  1. Press Ctrl+Shift+G to enable aggressive mode
  2. Click on any element that contains educational content
  3. Look for orange outline - indicates successful detection
  4. Click "I Need Help" to analyze the content

Step 2: Check Browser Console

Open browser console (F12) to see debug information:

  1. Press F12 to open developer tools
  2. Click on Console tab
  3. Click on educational content
  4. Look for debug messages showing what was detected

Step 3: Try Different Elements

Educational sites often have multiple ways to display the same content:

  • Click on the problem text (not just buttons)
  • Click on answer choices (A, B, C, D)
  • Click on input fields with math problems
  • Click on the question itself

Step 4: Use Text Selection as Fallback

If clicking doesn't work, try selecting text:

  1. Highlight the text you want to analyze
  2. Click "I Need Help" button
  3. Ask your question about the selected text

üéØ Common Educational Sites & Solutions:

Khan Academy

Problem: Uses complex Perseus widgets and custom elements

Solution:

  • Try clicking on the problem text itself
  • Use aggressive mode (Ctrl+Shift+G)
  • Select text manually if clicking doesn't work

Math Academy / Mathway

Problem: Uses custom math rendering and interactive elements

Solution:

  • Click on the math expression or problem
  • Try clicking on answer choices
  • Use aggressive mode for difficult elements

School Learning Management Systems

Problem: Custom interfaces and restricted JavaScript

Solution:

  • Enable aggressive mode
  • Try clicking on different parts of the problem
  • Use text selection as backup

Google Classroom / Canvas

Problem: Embedded content and iframe restrictions

Solution:

  • Click on the actual problem text
  • Try selecting text manually
  • Use aggressive mode for interactive elements

Check Extension Status

Make sure the extension is working:

  1. Go to chrome://extensions/
  2. Find "AI Tutor Assistant"
  3. Make sure it's enabled
  4. Check if there are any error messages

Test on Different Sites

Verify the extension works elsewhere:

  1. Try on a simple website first
  2. Test on our test pages
  3. Then try on the educational site

Clear and Reset

If nothing works, try resetting:

  1. Press Ctrl+Shift+D to clear everything
  2. Refresh the page
  3. Try again with aggressive mode
